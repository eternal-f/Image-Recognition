# Image-Recognition
This project mainly uses ModelArts of Huawei platform for image recognition. The main tasks include server setup, data annotation, model training, and analysis of prediction results. 
Some project conclusions are in Report/ModelArts.PDF. 
Some of the test results are shown in Picture/Detection_1.PNG and Picture/Detention_2.PNG.
